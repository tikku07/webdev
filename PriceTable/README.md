# Pricing Panel Project

A responsive and visually appealing **Pricing Panel Website** built using **HTML** and **CSS**. This project displays different pricing plans in a clean card-based layout, making it suitable for subscription-based services, SaaS platforms, or product pricing pages.

## Project Overview

The Pricing Panel Project is designed to showcase different subscription plans in a structured and user-friendly interface. It contains multiple pricing tiers with plan details, pricing information, and call-to-action buttons.

The webpage is fully responsive and adapts to different screen sizes using **CSS Flexbox** and **Media Queries**.

## Features

- Responsive pricing panel layout
- Three pricing plans:
  - Personal
  - Small Team
  - Enterprise
- Interactive hover effects on buttons
- Mobile-friendly responsive design
- Modern UI with card-based pricing structure
- Customized color palette and styling
- Icons/images for each pricing plan

## Technologies Used

- **HTML5** – Structure of the webpage
- **CSS3** – Styling and responsiveness
- **Google Fonts** – Typography enhancement
- **Flexbox** – Layout management
- **Media Queries** – Responsive design

## Responsive Design

The project is designed to work smoothly on both **mobile and desktop devices**.

- On smaller screens: Pricing plans appear in a **column layout**
- On larger screens (900px and above): Pricing plans appear in a **horizontal row layout**

## How to Run the Project

1. Download or clone the repository.
2. Ensure all project files are in the correct folder structure.
3. Open `index.html` in any web browser.

No additional installation is required.
